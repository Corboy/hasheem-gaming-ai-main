# HASHEEM GAMING

Digital game storefront built with Vite, React, TypeScript, and Tailwind CSS.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Test

```sh
npm run test
```
